# 📊 CSV 5GB Analysis (Benchmark)

هذا المشروع يهدف إلى اختبار أداء أدوات مختلفة للتعامل مع ملفات CSV ضخمة (5GB+).

---

## 🧰 المكتبات المستخدمة
- **Pandas** مع خاصية `chunksize`
- **Dask** لمعالجة البيانات الموزعة
- **ضغط الملفات (gzip)** لتقليل الحجم

---
